# Functional Storage Downgrade Fix (MC 1.20.1)
A small patch that fixes an item **overstack** issue in [Functional Storage](https://www.curseforge.com/minecraft/mc-mods/functional-storage) for Minecraft 1.20.1.
This patch fixes the conditions under which the Functional Storage iron downgrade can be applied and prevents the item overstack bug.

## Requirements
- Minecraft 1.20.1
- Functional Storage (1.20.1-1.2.10 or later)
