# Functional Storage OverStack Fix (MC 1.20.1)
A small patch that fixes an item **overstack** issue in [Functional Storage](https://www.curseforge.com/minecraft/mc-mods/functional-storage) for Minecraft 1.20.1.
This patch prevents items returned to the player from exceeding their maximum stack size when interacting with drawers. Any excess items are safely handled instead of creating illegal stacks.

## Requirements
- Minecraft 1.20.1
- Functional Storage (1.20.1-1.2.10 or later)
